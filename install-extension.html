NEXTAMP | EXTENSION GUIDE

วิธีติดตั้ง Nextamp Extension

เนื่องจากขณะนี้ Extension อยู่ในช่วงพัฒนา (Developer Preview) การติดตั้งจำเป็นต้องทำผ่าน Developer Mode บน Google Chrome


1

ดาวน์โหลดและแตกไฟล์

ดาวน์โหลดไฟล์ Extension (.zip) จากนั้นให้ทำการ Extract Files (แตกไฟล์) ออกมาเป็นโฟลเดอร์

2

ไปที่หน้าจัดการ Extension

เปิด Google Chrome แล้วพิมพ์ chrome://extensions ในช่อง URL แล้วกด Enter

3

เปิด Developer Mode

มองหาเมนูมุมขวาบนที่เขียนว่า Developer mode ให้ทำการกดเปิดสวิตช์ (ให้เป็นสีฟ้า)

4

Load Unpacked

1. คลิกปุ่ม Load unpacked ที่มุมซ้ายบน
2. เลือก โฟลเดอร์ ที่คุณแตกไฟล์ไว้ในขั้นตอนที่ 1

✓

เสร็จสิ้น!

ไอคอน Nextamp จะปรากฏที่แถบเครื่องมือ Chrome
คลิกเพื่อเริ่มใช้งานปรับแต่งเสียงบน YouTube หรือเว็บไซต์อื่นๆ ได้ทันที
